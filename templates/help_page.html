Useful information


This page acts as a device availability tool where users can add, remove and edit the devices used in the Software Testing department. Any device that is not locked, will reset its status and user at 7pm every night. The page auto-refreshes every 45 seconds so modifications must be made in that time frame. Adding, deleting and editing a device cannot all be done at the same time, they are sepparate actions. The user must choose to either add, remove or edit devices. The user may edit any number of devices but no modifications will take place unless they click "SUBMIT" at the end.

Adding a new device

To add a device, users will enter a device name or ip in the "Add new device" textbox:

Then click "SUBMIT".

Deleting a device

To delete a device, users will enter a device's ID in the "Delete device" textbox:

Then click "SUBMIT".

Last update

The "Last update" right under the "Delete device" label, shows the last time that anything was modified on the page.

Editing devices

The "Type" and "User" columns, are textboxes and users can enter the device's type and who is currently using the device. After any number of modifications to the already existing devices, the user must click "SUBMIT" in order for the changes to take place

The "Status" column, contains drop-down boxes where users can set a device as either "Busy" or "Available".

The "Lock" checkbox, will prevent a device from resetting its status and user at 7pm.

The "RESET FIELDS" button at the bottom of the page will reset the status and user of all devices, regardless of their lock status.
